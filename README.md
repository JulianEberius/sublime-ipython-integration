#Introduction:

This is simple plugin, which allows you to interact with ipython

#Features:

1. Evaluate code (selection, whole buffer) inside ipython
2. Provide object info inside sublimetext


#Requirements:

    - latest ipython
    - pyzmq
    - pyqt or pyside
